## 📖 About

Taryh (meaning *"History"* in Turkmen) is an interactive educational website with two mini-games and an informational section — all focused on the history and geography of Turkmenistan.

---

## 📂 Sections

### 1. 🏛️ Asgabat Binalary *(Ashgabat Buildings)*
An interactive **crossword puzzle** where players find the names of famous buildings in Ashgabat. Tests your knowledge of Turkmenistan's capital city architecture.

### 2. 🗺️ Gadymy Turkmenistan *(Ancient Turkmenistan)*
A **map-based geography game** where players click on a location and then select the correct region on the map to place it. A fun way to learn about ancient Turkmen lands.

### 3. 🎮 Oyunlar Barada *(About the Games)*
An informational page that explains how both games work and provides context about the history behind them.
